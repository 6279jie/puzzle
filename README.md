# 全能数独求解器

一个单文件的多变种数独求解器，适合直接在浏览器中打开使用，也可以部署到 GitHub Pages。

## 在线访问

- 在线体验：https://6279jie.github.io/puzzle/
- 仓库地址：https://github.com/6279jie/puzzle

## 功能概览

- 经典数独
- 对角线数独
- 无马数独
- 奇偶数独
- 连续数独
- 杀手数独
- 温度计数独
- 德耳语数独

## 使用方式

1. 直接用浏览器打开 `puzzle.html` 或 `index.html`
2. 选择数独类型
3. 输入题目或绘制约束
4. 点击求解相关按钮查看结果

## 项目特点

- 单文件实现，无需安装依赖
- 支持多种变体规则
- 支持约束绘制与规则说明
- 适合本地使用和静态网页部署

## 文件说明

- `index.html`：默认发布入口，适合 GitHub Pages
- `puzzle.html`：原始求解器文件
- `.nojekyll`：禁用 Jekyll 构建，按静态文件方式发布
